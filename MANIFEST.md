# Package contents

- `CLAUDE_CLI_PROMPT.md` — ready-to-paste Claude CLI commission.
- `ARCHITECTURE.md` — decision-ready minimal architecture.
- `intent/` — operator-authored product, building, actor, and machine declarations.
- `observed/raw/` — unmodified repository census outputs supplied by the operator.
- `observed/summary/` — compact active-view summaries.
- `decisions/` — explicit intent/reality reconciliation records.
- `src/` — small validation and Mermaid/Markdown rendering reference implementation.
- `generated/` — sample views produced by the reference implementation.

The code is a blueprint checker/renderer only. It is not a production backend implementation.
